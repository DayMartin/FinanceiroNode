Software Web para gerenciamento de: 
  1. Compra
  2. Venda
  3. Gerenciamento de Estoque
  4. Contas a pagar
  5. Contas a receber
  6. Relatórios
  7. Comissão de vendedor
  8. Compras de cliente

Back: TypeScript, NodeJS, Express, JWT

Front: React, Axios

Banco: MySQL
